# Huffman Coding

> The purpose of this is solely educational and therefore it is not advised to use it in any practical scenario.

This repository contains C# implementation of [Huffman coding](https://en.wikipedia.org/wiki/Huffman_coding) algorithm where every 4096 bytes of data are encoded separately. The application is designed to take file as an input and produce a file as the output if filename is provided. Otherwise it is dumped to the stdout.
